🚀 FastAPI Project

This project showcases the features and capabilities of the FastAPI framework, ideal for building modern, high-performance web APIs.

🔐 Authentication

  • User registration and JWT-based login
  • Token refresh and secure password hashing

👤 User Management

  • User account creation, profile view and update
  • Admin-only field access (last login, user ID, role)
  • Role-based data access with query optimization
  • Limited visibility for normal users

📝 Todo Management

  • CRUD operations for authenticated users
  • Pydantic-based input validation
  • PostgreSQL integration and custom error handling

⚙️ Requirements

  • Python 3.8+
  • uv (install: pip install uv)
  • Uvicorn

📦 Installation

git clone git@github.com:DhruvSimform/fastapi_project.git
cd fastapi_project
uv run uvicorn src.main:app --reload

🚀 Run the Application
